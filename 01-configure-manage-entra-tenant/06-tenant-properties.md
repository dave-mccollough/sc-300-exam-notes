# Tenant Properties

- Azure Portal > Entra ID > Properties tab
  - Shows tenant properties
  - Security defaults
  - Manage conditional access

- User settings
  - Azure Portal > Entra ID > Users > User settings
    - Default user role permissions
    - Guest user settings
    - Adminstrator Center
    - LinkedIn integration
    - Show keep user signed in
    - External users and collaboration settings
    - User features

- Group settings
  - Azure Portal > Entra ID > Groups > Settings > General
    - Self service group management
    - Security groups
    - M365 groups
    - Directory wide groups

- Group Expiration
  - Azure Portal > Entra ID > Groups > Settings > Expiration
  - You can create groups that expire

- Naming policy
  - Azure Portal > Entra ID > Groups > Settings > Naming policy
    - Blocked words
    - Group naming policy

- Device Settings
  - Azure Portal > Entra ID > Manage > Devices
  - Device Settings
    - Users can join devices to Entra ID
    - Users can register devices to Entra ID
      - Disabled with Intune enabled
    - Require MFA to join or register devices to Entra ID
    - Local admin settings
    - Enable LAPS
    - Other settings
      - Restrict users from obtaining their own bitlocker keys
